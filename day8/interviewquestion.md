Interview Questions
1. What is JavaScript?

JavaScript is a programming language used to add interactivity and dynamic behavior to web pages. It runs in the browser and can also run on servers using Node.js.

2. Why do we use JavaScript?

We use JavaScript to:

Respond to user actions
Validate forms
Manipulate web pages
Fetch data from servers
Build backend applications with Node.js
3. What is console.log()?

console.log() is a JavaScript method used to print information to the browser's developer console.

Example:

console.log("Hello World");
4. What are the three core technologies of the web?
HTML – Structure
CSS – Styling
JavaScript – Functionality
5. Why is the <script> tag usually placed before </body>?

Because it allows the HTML to load first, ensuring that JavaScript can safely interact with the page elements after they have been created.
